#EAGLEをインストールする

[EagleのHP](http://www.cadsoftusa.com/download-eagle/?language=en)

自分のOSに合わせてEagleをインストール。

英語に負けず起動する。
---
#EAGLEでプロジェクトをつくる

ツールバーから

File -> New -> Project

を選択して新しいプロジェクトを作る。名前は適当に…

次に作ったプロジェクトを右クリックして

Edit -> New -> Schematic

をクリックすると新しい回路図を作る画面が表示される。
